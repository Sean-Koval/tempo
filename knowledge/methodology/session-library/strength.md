# Session Library — Strength

## Strength

### `strength_foundation`
- **Purpose:** anatomical adaptation, injury prevention
- **Duration:** 45 min
- **TSS:** 20 (est.)
- **Structure:** 3×8–10 squat, deadlift, split squat, row, core

### `strength_maintenance`
- **Purpose:** preserve strength during build/peak
- **Duration:** 30 min
- **TSS:** 15 (est.)
- **Structure:** 2×5 heavy (compound), core, done

### `strength_hypertrophy_block`
- **Purpose:** build cross-sectional area + work capacity in early base; foundation for later max-strength
- **Duration:** 50–60 min
- **TSS:** 25 (est.)
- **Structure:** 2 main lower (squat variant + deadlift/RDL) + 1 upper pull + 1 single-leg, 3–4×8–12 reps @ 65–75%1RM, 90s rest. Finish: core + 1 plyo primer (e.g., 3×3 box step-ups). Run 3–6 weeks in `prep_*` / early `base_*` phases.
- **Notes:** stop adding load if next-day run/bike Z2 HR drifts > 5 bpm at same pace — bias under-recovery R-2/R-11.

### `strength_intensification_block`
- **Purpose:** convert hypertrophy into maximal strength (Rønnestad model: heavy, low rep, full recovery → economy gains)
- **Duration:** 45–55 min
- **TSS:** 20 (est.)
- **Structure:** 2 main lifts (back squat + trap-bar/conventional deadlift), 4×4–6 reps @ 80–90%1RM, 2.5–3 min rest; 1 accessory pull (3×6); core 5 min. No new movements introduced here. Run 4–8 weeks in `base_durability_*` / early `build_*`.
- **Notes:** schedule ≥ 24 h before hard run; never within 24 h of a long-run / race-pace bike. Cap session at 60 min — fatigue accumulation is the limiter, not volume.

### `strength_realization_block`
- **Purpose:** in-season maintenance + neuromuscular sharpness during build/peak
- **Duration:** 25–35 min
- **TSS:** 12 (est.)
- **Structure:** 1 main lift (heavy squat or deadlift) 3×3 @ 85–92%1RM, 3 min rest; 1 explosive primer (3×3 jump squat or 3×5 box jump); core 5 min. Run during `build_*` / `peak_*`.
- **Notes:** drop to `strength_maintenance` (or skip) in race week. Prioritize bar-speed quality — if rep speed visibly slows, end the session.
