# Strength Program — Cycling-Led Build (BSI Recovery)

Authored by Sean 2026-05-01, anchored to the current cycling-primary block while left-tibia BSI heals. 4 strength touches/week (3 full × 45 min + 1 mini × 20 min). Goals: maintain upper body, build lower-body strength for cycling power and run return, protect the healing tibia (graded calf loading on the left).

This file is **athlete state**, not methodology. When a plan exists, the bootstrap will fold this into a plan-fragment YAML under `plans/<plan-id>/fragments/strength-cycling-build.yaml` (per `tempo-0e3` `/goal-research` schema) so the composer can budget it against weekly TSS via R-20.

## Weekly cadence

| Day | Session | Duration |
|---|---|---|
| Mon | Day A — Posterior Chain + Single Leg (Lower) | 45 min |
| Tue (PM) | Day D — Mini Upper Maintenance | 20 min |
| Wed | Day B — Upper Body Push/Pull + Core | 45 min |
| Fri | Day C — Hip Stability + Cycling Power + Core | 40 min |

Day A and Day C are both lower-focused but hit different patterns — A is posterior/single-leg dominant, C is hip stability and bilateral cycling power. Together = 2 true lower sessions/week.

## Progression rules

- **Left calf raise (Day A)** — start 2 × 10 BW, add 1 rep/week to a target of 3 × 15 BW before adding load. Gated by zero-pain criterion at the BSI site.
- **Single-leg RDL (Day A)** — progress load only when left side matches right load × reps cleanly for 2 sessions in a row.
- **Hip thrust, Bulgarian split, goblet squat** — linear-progression by 2.5 kg/week or +1 rep, whichever sustains form.
- All upper-body lifts: maintain working load; this is a maintenance phase, not hypertrophy push.

## Stop / back-off rules

- Any reproducible pain at BSI site during or after a session → drop to bilateral-only the next session, log to `journal/`, flag in next morning check-in.
- DOMS > 48 hr in any major group → cut next session's volume by 30%, keep the lifts but trim sets.
- Cycling FTP test ≥ 7 days out → drop strength volume 30% in the test week.
